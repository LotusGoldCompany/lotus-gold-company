# FAQ Section Model Scaling

The objective is to increase the size of the FAQ model image across all device sizes to make her a more prominent visual element.

## Proposed Changes

### [MODIFY] [page.jsx](file:///c:/Users/JUDE/OneDrive/Desktop/Lotus%20Gold%20Company/lotus-gold-company/app/page.jsx)

- **Container Width**: Increase `lg:w-[35%]` to `lg:w-[40%]`.
- **Image Height (Mobile/Tablet)**: Increase `h-[400px]` to `h-[500px]` and `md:h-[500px]` to `md:h-[600px]`.
- **Scaling Effect**: The model will take up more horizontal space on desktop and more vertical space on mobile/tablet.

## Verification Plan

- View on desktop. The model should occupy more of the right-hand side of the screen.
